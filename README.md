# A Angular web application for Dev Book Store
Explore Angular 2 Features with My Sample Book Store Project

To see the project:
- Clone/download it
- Use ```npm install```
- Use ```npm start```

## This application includes examples of major features of Angular including:

- NgModules
- Components
- Data Binding
- Directives
- Pipes
- Interface
- Lifecycle Hooks
- Input & Output
- Services
- HTTP and Observables
- Routing
